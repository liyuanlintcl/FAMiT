### Quick Start

#### Environment Setup

```shell
python3 -m venv <name>
source <name>/bin/activate
pip install -r ./requirements.txt  
```

#### Run Pysa 

```shell
pyre analyze --no-verify --save-results-to ./bench/pysa-runs
```

#### Run FAMiT

You should use your own api-key before run FAMiT.

```shell
python3 ./analysis.py
```
